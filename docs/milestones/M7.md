# M7 交付说明（userspace）

## 范围

- 最小 shell：`echo` / `ls` / `cat`
- 支持重定向 `>` 与单级管道 `|`
- 用户态网络工具源码：`ping` / `http_server` / `http_client`
- 集成测试：boot -> shell script -> 命令校验

## 代码文件

- `include/nm/shell.h`
- `include/nm/userspace.h`
- `userspace/shell.c`
- `kernel/userspace/init.c`
- `tests/unit/test_shell.c`
- `tests/integration/test_boot_shell.c`

## 关键函数签名

- `int shell_execute_line(const char *line, char *out, size_t out_cap);`
- `int shell_run_script(const char *script, char *out, size_t out_cap);`
- `void userspace_init(void);`

make clean all
